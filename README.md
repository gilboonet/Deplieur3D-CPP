#  Deplieur3D-CPP
Volume unfolder with 3D UI to select faces

It's a Qt C++ project, built with CMAKE on QT Creator (14.0.2)
using Desktop MinGW kit, MSVC Qt 6.8.0 kit or WebAssembly Qt 6.8.0 kit.
