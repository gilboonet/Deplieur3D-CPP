#  Deplieur3D-CPP
Déplie un volume en sélectionnant des pièces par coloration des faces grâce à une UI en 3D

C'est un projet Qt C++, construit (CMAKE) avec Qt Creator (15.0.1)
(kit Desktop Qt 6.8.2 ou WebAssembly Qt 6.8.2)
